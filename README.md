# mujoco_irb120
Mujoco Simulation of ABB irb120 6dof manipulator for 3D CoM Estimation

## Installing
Using python 3.10+ import all req'd packages using requirements.txt

The mujoco object models are from https://github.com/vikashplus/object_sim.git and must be
installed by running: 

git clone --recurse-submodules [THIS REPO]
